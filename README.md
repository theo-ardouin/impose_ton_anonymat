- `!impose image`
- `!impose start [TIME...]`
- `!impose stop`
- `!impose test`

- `PYTHONPATH=$(pwd) ./scripts/dump.py <DIR>`
- `PYTHONPATH=$(pwd) ./scripts/permissions.py <USER> <SCOPE...>`
